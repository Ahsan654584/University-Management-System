# 🎓 University Management System

A comprehensive University Management System (UMS) designed to streamline and automate academic and administrative processes. This system handles student records, course registration, faculty management, grades, and more.

---

## 📌 Features

- **Student Module**
  - Registration and profile management
  - Course enrollment
  - Grade viewing

- **Faculty Module**
  - Course assignment
  - Student evaluation and grading
  - Timetable management

- **Admin Module**
  - Add/edit/delete students, faculty, courses, and departments
  - Generate reports
  - Manage academic calendar

- **Authentication**
  - Secure login system for students, faculty, and admins
  - Role-based access control

---

## 💻 Technologies Used

- **Frontend**: HTML, CSS
- **Backend**:  Java 
- **Database**: MySQL 
- **Tools**: Git, GitHub, VS Code
